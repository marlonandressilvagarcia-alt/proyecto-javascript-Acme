# proyecto-javascript-Acme
Proyecto Acme - Marlon Andres Silva Garcia
Es una aplicación para ayudar a la empresa Acme a organizar sus cosas. Básicamente, sirve para tres cosas principales: registrar usuarios, llevar el control de lo que hay en bodega (inventario) y fabricar productos usando recetas (producción).

Lo que se usó
HTML para las pantallas.

CSS para los colores y el diseño.

JavaScript para que el programa haga los cálculos y guarde los datos.

Firebase para guardar toda la información en la nube.

¿Cómo se usa?
Abres el archivo index.html en el navegador.

Entras con tu usuario y contraseña.

Arriba verás unos botones para moverte entre Usuarios, Inventario y Producción.

Los módulos
Login: Es la pantalla inicial para entrar al sistema.

Usuarios: Aquí puedes agregar o borrar a las personas que usan el programa.

Inventario: Aquí metes las materias primas o productos. También puedes ver cuánto stock queda y buscar cosas rápido.

Producción: Aquí eliges qué quieres fabricar. El sistema revisa si tienes materiales suficientes y si alcanza, los descuenta y agrega el producto terminado.
